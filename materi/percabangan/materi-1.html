Percabangan If

Percabangan if adalah control flow statement yang paling dasar di dalam pemrograman Java.


Struktur penulisan IF: 
if( suatu_kondisi ) {
    // lakukan sesuatu kalau kondisi benar
    // Lakukan ini juga
}
Contoh: 
package id.ac.ub.filkom.codemaniac.materijava;
public class PercabanganIf {
    public static void main(String[] args) {
        // membuat variabel belanja dan scanner
        int belanja = 0;
        Scanner scan = new Scanner(System.in);

        // mengambil input
        System.out.print("Total Belanjaan: Rp ");
        belanja = scan.nextInt();

        // cek apakah dia belanja di atas 100000
        if ( belanja > 100000 ) {
            System.out.println("Selamat, anda mendapatkan hadiah!");
        }

        System.out.println("Terima kasih telah berbelanja di Aneka Swalayan");
    }
}
