Percabangan If

Percabangan if adalah control flow statement yang paling dasar di dalam pemrograman Java.


Struktur penulisan IF: 
if( suatu_kondisi ) {
    // lakukan sesuatu kalau kondisi benar
    // Lakukan ini juga
}
Contoh: 
package id.ac.ub.filkom.codemaniac.materijava;
public class PercabanganIf {
    public static void main(String[] args) {
        // membuat variabel belanja dan scanner
        int belanja = 0;
        Scanner scan = new Scanner(System.in);

        System.out.println("== Supermarket Griya UB ==");
        // mengambil input
        System.out.print("Total Belanjaan: Rp ");
        // user memasukkan nilai total belanjaan Rp 150.000
        belanja = scan.nextInt();

        System.out.println("--------------------------");
        // cek apakah dia belanja di atas 100000
        if ( belanja > 100000 ) {
            System.out.println("Selamat, anda mendapatkan hadiah!");
        }

        System.out.println("Terima kasih telah berbelanja di Supermarket Griya UB ");
    }
}
Output: 
== Supermarket Griya UB ==
Total Belanjaan: Rp 150000
--------------------------
Selamat, anda mendapatkan hadiah!
Terima kasih telah berbelanja di Supermarket Griya UB
