Input Output

Pemrograman Java terdiri dari 3 komponen dasar yaitu Input, Proses dan Output.

  1. Input : Nilai yang dimasukkan.
  2. Proses : Serangkaian langkah yang dilakukan untuk mengelola input yang diberikan.
  3. Output : Menampilkan hasil olah data.
Pada bahasa pemrograman Java, Terdapat beberapa library untuk membantu pengambilan input berbasis teks, yakni:
  1. Class Bufferedreader
  2. Class Scanner
Perbedaan BufferReader dengan Scanner terlihat dari fungsi atau method yang dipakai. Scanner menggunakan next(), sedangkan BufferReader menggunakan readLine(). Lalu untuk tipe data integer, BufferReader menggunakan fungsi read() saja.
