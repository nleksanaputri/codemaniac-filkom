Mengambil Input dengan Scanner

Scanner merupakan kelas yang menyediakan fungsi-fungsi untuk membaca dan mengambil input dari pengguna. Scanner memiliki kemudahan yang dapat membaca teks, baik yang memiliki tipe data primitif maupun string. Cara menggunakan scanner dalam sebuah program yakni dengan mengimpornya ke dalam kode menggunakan import java.util.Scanner;

Penggunakan fungsi untuk mengambil data bergantung pada tipe data yang digunakan. Apabila tipe datanya adalah String, maka fungsi atau method yang dipakai adalah nextLine(). Namun, jika tipe data yang digunakan adalah integer, maka menggunakan nextInt(), dan sebagainya.

Contoh: 

package id.ac.ub.filkom.codemaniac.materijava;
import java.util.Scanner;
public class InputScanner {
    public static void main(String[] args) {
        // deklarasi variabel
        String kata;

        // membuat scanner baru
        Scanner keyboard = new Scanner(System.in);

        // Tampilkan output ke user
        System.out.print("Masukan Sebuah Kata : ");

        // menggunakan scanner dan menyimpan apa yang diketik di variabel nama
        kata = keyboard.nextLine();

        // Menampilkan apa yang sudah simpan di variabel
        System.out.println("Kata Yang Anda Masukan Adalah : " + kata);
    }
}
Output: 
Masukan Sebuah Kata : Hello World!
Kata Yang Anda Masukan Adalah : Hello World!
