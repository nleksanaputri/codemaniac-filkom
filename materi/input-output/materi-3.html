Menampilkan Output

Berikut adalah fungsi yang digunakan untuk menampilkan output dari program yang sudah disediakan oleh Java:

  1. Fungsi System.out.print()
  2. Fungsi System.out.println()
Fungsi print() akan menampilkan teks apa adanya. Sedangkan println() akan menampilkan teks dengan tambahan baris baru.

Contoh: 
package id.ac.ub.filkom.codemaniac.materijava;
public class Output {
    public static void main(String[] args) {
        System.out.println("Dicetak pakai println()");
        System.out.println("Ini juga dicetak pakai println()");
        System.out.print("Ini dicetak dengan print()");
        System.out.print(" dan ini juga dicetak dengan print()");
    }
}
Output: 
Dicetak pakai println()
Ini juga dicetak pakai println()
Ini dicetak dengan print() dan ini juga dicetak dengan print()
