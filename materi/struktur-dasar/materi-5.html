Case Sensitive

Java bersifat Case Sensitive, artinya penulisan dengan huruf besar atau kapital dan huruf kecil akan dibedakan. Contoh: 

package id.ac.ub.filkom.codemaniac.materijava;
public class CaseSensitive {
    public static void main(String[] args) {
        String nama = "codemaniac";
        String Nama = "Code Maniac";
        String NAMA = "Code MANIAC";

        System.out.println(nama);
        System.out.println(Nama);
        System.out.println(NAMA);
    }
}
Output: 
codemaniac
Code Maniac
Code MANIAC

Tiga variabel yang sudah dideklarasikan di atas merupakan tiga variabel yang berbeda, meskipun sama-sama memiliki nama sama.
