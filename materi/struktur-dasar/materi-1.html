Deklarasi Package

Package adalah folder yang berisi kumpulan program Java. Pada umumnya deklarasi package dilakukan ketika membuat program atau aplikasi yang berukuran besar. Langkah penamaan package:

  • Umumnya menggunakan nama domain institusi (dengan penulisan di balik) pembuat program tersebut.
  • Sertakan nama program/aplikasi. Ikuti dengan nama modul-modulnya.
  • Beri pemisah dengan tanda titik.
Contoh:
package com.google.program;

pada contoh di atas com.google adalah nama domain dari Google (www.google.com).

Impor Library

Library adalah kumpulan kelas dan fungsi yang dapat digunakan untuk membuat program. Pada bagian ini, kita melakukan impor library yang diperlukan pada program. Contoh:

import java.util.Scanner;

Pada contoh di atas kita mengimpor class Scanner dari package java.util.

Class

Java merupakan bahasa pemrograman yang menggunakan paradigma OOP (Object Oriented Programming). Maka dari itu setiap program yang dibuat harus dibungkus di dalam class agar nanti bisa dibuat menjadi objek. Contoh: 

public class NamaProgram {
    public static void main(String args[]){
        System.out.println("Hello World");
    }
}

Blok class dibuka dengan tanda kurung kurawal { kemudian ditutup atau diakhiri dengan tanda kurung kurawal }. Pada contoh di atas, terdapat method main(). Selain method, di dalam blok class kita juga dapat mengisinya dengan fungsi-fungsi dan juga variabel.

Metod Main

Method main() atau fungsi main() merupakan blok program yang akan dieksekusi pertama kali. Method main() wajib kita buat ketika membuat program di Java. Karena jika tidak dibuat maka programnya tidak akan bisa dieksekusi. Contoh: 

public static void main(String args[]){
    System.out.println("Hello World");
}

Method main() memiliki parameter args[], yakni parameter yang akan menyimpan sebuah nilai dari argumen di command line. Kemudian di dalam method main() terdapat statement atau fungsi. Pada contoh di atas salah satu contoh statement yang diberikan adalah System.out.println("Hello World");, ini adalah fungsi untuk menampilkan teks ke layar monitor.

Statement dan Ekspresi pada Java

Statement dan ekspresi adalah bagian terkecil dalam sebuah program. Setiap statement dan ekspresi yang ada di Java, harus diakhiri dengan titik koma (;). Contoh: 

System.out.println("Hello World");
System.out.println("Code Maniac");
int x = 3;
int y = 8;
int z = x + y;

Pada contoh di atas, kita memberi perintah kepada komputer untuk menampilkan teks "Hello World", dan "Code Maniac". Kemudian kita memberikan perintah untuk menghitung nilai x + y.
