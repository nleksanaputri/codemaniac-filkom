Operasi Unary

Operasi Unary merupakan operator yang melakukan berbagai operasi aritmatik yang hanya melibatkan satu operand. Operasi-operasi tersebut dapat berupa penambahan, pengurangan atau membalik suatu nilai dari bolean. Berikut adalah yang termasuk Operator Unary:

Contoh: 
package id.ac.ub.filkom.codemaniac.materijava;
public class OperasiUnary {
    public static void main(String args[]) {
        System.out.println("Operator Unary Plus");
        int nilaiAwal = 12;
        int hasil = +nilaiAwal;
        System.out.println("Hasil +12 = " + hasil);
        System.out.println();

        System.out.println("Operator Unary Minus");
        int nilaiAwal2 = 18;
        nilaiAwal2 = -nilaiAwal2;
        System.out.println("Hasil -18 = " + nilaiAwal2);
        System.out.println();

        System.out.println("Operator peningkatan nilai sebesar 1 point");
        int nilaiAwal3 = 25;
        nilaiAwal3++;
        System.out.println("Hasil 25++ = " + nilaiAwal3);
        System.out.println();

        System.out.println("Operator pengurangan nilai sebesar 1 point");
        int nilaiAwal4 = 13;
        nilaiAwal4--;
        System.out.println("Hasil 13-- = " + nilaiAwal4);
        System.out.println();

        System.out.println("Operator komplemen logika");
        boolean sukses = false; //Nilai sukses adalah false
        System.out.println("Hasil !false = " + !sukses);
        System.out.println();
    }
}
Output: 
Operator Unary Plus
Hasil +12 = 12

Operator Unary Minus
Hasil -18 = -18

Operator peningkatan nilai sebesar 1 point
Hasil 25++ = 26

Operator pengurangan nilai sebesar 1 point
Hasil 13-- = 12

Operator komplemen logika
Hasil !false = true
