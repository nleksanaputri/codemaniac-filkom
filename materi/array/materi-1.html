Deklarasi Array

Array adalah sebuah variabel yang bisa digunakan untuk menyimpan kumpulan data lebih dari satu dengan tipe data yang sama. Array memiliki jumlah data yang fixed (tetap). Array menggunakan index untuk memudahkan akses terhadap data yang disimpan. Index array dimulai dari 0.
Berikut adalah cara untuk mendeklarasikan suatu array:

  • Gunakan kurung siku [] untuk membuat array.
  • Kurung siku bisa diletakkan setelah tipe data atau nama array.
  • Tentukan length (panjang) array. Definisikan panjang dengan menuliskan jumlahnya dalam nominal integer di dalam tanda [].
Contoh: 
void deklarasiArray(){
    // cara pertama
    String[] namaArray;
    
    // cara kedua
    String namaArray [];
    
    // cara ketiga
    // panjang array adalah 5
    String[] namaArray = new String[5];
    String namaArray [] = new String[5];
}
