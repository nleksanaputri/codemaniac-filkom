IndexOutOfBounds

IndexOutOfBounds adalah error yang disebabkan oleh percobaan saat melakukan akses indeks yang tidak dimiliki oleh array. Contoh: 

package id.ac.ub.filkom.codemaniac.materijava;
public class IndexOutOfBounds {
    public static void main(String[] args) {
        int[] arrA = {1, 2, 3, 4};
        System.out.println(arrA[0]);
        System.out.println(arrA[1]);
        System.out.println(arrA[2]);

        // Akses indeks ke 4 akan error karena jumlah elemen array hanya 4 dan index array dimulai dari 0
        System.out.println(arrA[4]);
    }
}
Output: 
1
2
3
Exception in thread "main" java.lang.ArrayIndexOutOfBoundsException: Index 4 out of bounds for length 4
