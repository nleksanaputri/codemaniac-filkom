Tipe Data Reference

Tipe data reference merupakan sebuah tipe data yang merujuk ke sebuah objek atau instance dari sebuah class. String adalah salah satu tipe data yang termasuk ke dalam tipe data reference. Tipe data string menunjuk ke instance dari class java.lang.String.

String

String merupakan sebuah tipe data reference. Tipe data string diperlakukan sebagai sebuah objek. String merupakan sebuah class yang terdapat dalam library Java dan digunakan untuk memanipulasi karakter.
Contoh: 

String greeting = "Hello World!";

Pada contoh di atas, konsol akan menampilkan teks Hello World!

Mengetahui Panjang String

Method length() merupakan method yang digunakan untuk mengetahui panjang dari sebuah string. Fungsi tersebut akan mengembalikan atau menghasilkan sejumlah karakter dari string.
Contoh: 

package id.ac.ub.filkom.codemaniac.materijava;
public class PanjangString {
    public static void main(String[] args) {
        String name = "Code Maniac";
        int length = name.length();
        System.out.println(length);
    }
}
Output: 
11

Fungsi length() pada kode di atas akan menghitung jumlah karakter dari variabel name.

Mengambil Karakter Dari Sebuah String

Method charAt(int index) merupakan method yang digunakan untuk mengambil sebuah karakter secara spesifik dari sebuah String dalam kelas String.
Contoh: 

package id.ac.ub.filkom.codemaniac.materijava;
public class KarakterString {
    public static void main(String[] args) {
        String name = "Code Maniac";
        char result = name.charAt(7);
        System.out.println(result);
    }
}
Output: 
n

Fungsi charAt(7) pada kode di atas akan menampilkan karakter urutan ke-7 dari teks tersebut, yaitu "n".
