Tipe Data Primitif

Tipe data primitif adalah tipe data standar yang tidak diturunkan dari objek manapun. Tipe data primitif telah ditentukan dengan kata kuncinya masing-masing. Terdapat 8 (delapan) tipe data primitif yang Java dukung, antara lain:

Boolean

Boolean merupakan sebuah tipe data yang memiliki 2 (dua) macam nilai, yakni true dan false. Tipe data float memiliki nilai default false.
Contoh: 

boolean value = true;
boolean anotherValue = false;

Char

Char merupakan sebuah tipe data yang bisa digunakan untuk menampung karakter. Nilai karakter dibungkus di dalam tanda peti tunggal (' ').
Contoh: 

char item = 'A'

Bilangan Bulat (Integer)

Tipe bilangan bulat (Integer) adalah tipe data numerik yang biasa digunakan oleh bilangan bulat, seperti 1, 27, 100, dan lain-lain. Bilangan bulat juga mengenal nilai positif dan negatif (signed number). Tipe data numerik yang termasuk ke dalam bilangan bulat adalah sebagai berikut:


Byte

Byte adalah tipe data integer 8 bit yang digunakan untuk menampung angka dalam range yang kecil. Nilai minimum dan maksimum dari tipe data byte adalah -128, dan 127. Tipe data byte memiliki nilai default 0.
Contoh: 

byte value = 10;
byte anotherValue = -10;

Short

Short merupakan tipe data integer 16 bit yang digunakan untuk menampung angka dalam range menengah, yaitu antara -32,768 sampai 32,767. Tipe data short memiliki nilai default 0.
Contoh: 

short value = 15000;
short anotherValue = -20000;

Int

Int merupakan tipe data integer 32 bit yang digunakan untuk menyimpan angka dalam range cukup besar, yakni antara -2,147,483,648 sampai 2,147,483,647. Tipe data int memiliki nilai default 0.
Contoh: 

int value = 150000;
int anotherValue = -200000;

Long

Long merupakan tipe data integer yang lebih besar, yakni 64 bit dan bisa digunakan untuk menyimpan angka dengan range antara -9,223,372,036,854,775,808 sampai 9,223,372,036,854,775,807. Tipe data long memiliki nilai default 0L.
Contoh: 

long value = 150000L;
long anotherValue = -200000L;


Bilangan Pecahan (Floating Point)

Tipe bilangan pecahan atau floating point adalah bilangan yang menangani bilangan desimal atau perhitungan secara detail. Terdapat dua tipe pada bilangan pecahan ini yakni:


Float

Float merupakan sebuah tipe data yang bisa digunakan untuk menampung angka desimal. Tipe data float memiliki nilai default 0.0f.
Contoh: 

float value = 3.5f;

Double

Double merupakan sebuah tipe data yang bisa menampung angka desimal, dan memiliki kapasitas yang lebih besar daripada float. Tipe data float memiliki nilai default 0.0d.
Contoh: 

double value = 5.0;
